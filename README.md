# NogizakaBlog

Get the number of comments and articles per month from Nogizaka46's blog

## Installation

Add this line to your application's Gemfile:

    gem 'nogizaka_blog'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nogizaka_blog

## Usage

	$ ruby nogizaka_blog -m 201405
